🩺 Disease Prediction System using Machine Learning
📌 Overview
This project is a Machine Learning-based Disease Prediction System that predicts possible diseases based on user-input symptoms. It uses multiple classification algorithms, compares their performance, and selects the best model for accurate predictions.

🎯 Features
Predicts diseases based on symptoms

Uses multiple ML algorithms for comparison

Selects the best-performing model automatically

Handles spelling errors using fuzzy matching

Provides data visualization (graphs & heatmaps)

User-friendly symptom input system

🛠️ Technologies Used
Python

Pandas, NumPy

Scikit-learn

Matplotlib, Seaborn

Difflib (for fuzzy matching)

Jupyter Notebook

🤖 Machine Learning Models Used
Decision Tree

Naive Bayes

K-Nearest Neighbors (KNN)

Logistic Regression

📂 Dataset
Contains symptoms as input features

Diseases as target output

Stored in CSV format

⚙️ How It Works
Load and preprocess the dataset

Train multiple machine learning models

Evaluate models using accuracy

Select the best model

Take user input (symptoms)

Convert input into feature vector

Predict disease

📊 Output
Displays the predicted disease based on input symptoms

Shows accuracy comparison of different models

Includes visualizations like graphs and heatmaps

🚧 Limitations
Depends on dataset quality

Cannot use heavy models like Random Forest due to memory constraints

Predicts only one most probable disease

🚀 Future Improvements
Add more advanced models (if resources allow)

Use larger and real-time datasets

Build a web or mobile application interface

Provide multiple disease suggestions with probability
